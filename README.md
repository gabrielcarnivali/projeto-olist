Projeto para análise de dados

Neste projeto eu tive o objetivo de aplicar os conhecimentos de exploração, limpeza análise de dados. Utilizei um dataset do Kaggle para a atividade.

Meu foco era chegar a um resultado final claro e útil, e validando meu teste de conhecimento, consegui formular algumas perguntas de raciocínio próprio:

1. Prazo médio de entrega ao cliente
2. Prazo médio entre aprovação da compra e saída para entrega
3. Média de antecedência em relação à data prevista de entrega
4. Taxa de atraso entre pedidos entregues
5. Média de dias de atraso, quando atrasa

Utilizei Python com as bibliotecas pandas e matplotlib, no formato de Jupyter Notebook.

Dataset utilizado da plataforma Kaggle: 
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Fiquei satisfeito com o resultado quando consegui analisar todas as informações coletadas, e validar meus conhecimentos em lógica de programação, tratativa e análise de dados.

Esse é o meu primeiro projeto que vou subir ao GitHub, para a exposição dos meus conhecimentos.
